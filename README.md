# AutomatedEmployeeManagementSystem

AEMS is a Next.js-based Automated Employee Management System prototype for managing employee records, payroll visibility, appraisals, renewals, projects, and role-based dashboards for Owner, HR, Manager, and Employee users.

## Getting Started

Run the local development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Current Scope

- Branded landing page and demo login
- Role-based portal shell
- Owner, HR, Manager, and Employee dashboard views
- Employee profile, payroll, projects, departments, and reports pages
- Interactive dashboard drill-down cards
- Floating collapsible sidebar

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
